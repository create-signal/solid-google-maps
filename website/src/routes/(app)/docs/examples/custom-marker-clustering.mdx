## About

Custom Marker Clustering requires three additional dependencies - [supercluster](https://www.npmjs.com/package/supercluster), [geojson](https://www.npmjs.com/package/geojson) and [@solid-primitives/keyed](https://www.npmjs.com/package/@solid-primitives/keyed).

## Installation

```bash
npm i supercluster @types/supercluster geojson @types/geojson @solid-primitives/keyed
```
