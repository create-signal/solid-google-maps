## Event Types

All Map events extend the type `MapEvent` and have the following properties:

| Property    | Type/Description                                                                                                               |
| ----------- | ------------------------------------------------------------------------------------------------------------------------------ |
| `type`      | `string`<br />The type of the event.                                                                                           |
| `map`       | `google.maps.Map`<br />The map instance that the event was fired on.                                                           |
| `stoppable` | `boolean`<br />Indicates if the event is stoppable.                                                                            |
| `stop`      | `() => void`<br />A function that stops the event from propagating further.                                                    |
| `domEvent`  | `MouseEvent \| TouchEvent \| PointerEvent \| KeyboardEvent \| Event`<br />The original DOM event that triggered the map event. |

### MapMouseEvent

Contains a detail property with the latLng and placeId of the clicked location

```typescript
{
  type: string
  map: google.maps.Map
  detail: {
    latLng: google.maps.LatLngLiteral | null;
    placeId: string | null;
  }
  stoppable: boolean
  stop: () => void
  domEvent?: MouseEvent | TouchEvent | PointerEvent | KeyboardEvent | Event
}
```

### MapCameraChangedEvent

Contains a detail property with the position of the camera

```typescript
{
  type: string
  map: google.maps.Map
  detail: {
    center: google.maps.LatLngLiteral
    bounds: google.maps.LatLngBoundsLiteral
    zoom: number
    heading: number
    tilt: number
  };
  stoppable: boolean
  stop: () => void
  domEvent?: MouseEvent | TouchEvent | PointerEvent | KeyboardEvent | Event
}
```
