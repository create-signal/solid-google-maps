## API Reference

### Props

| Property      | Default | Type/Description                                                             |
| ------------- | ------- | ---------------------------------------------------------------------------- |
| `background`  | -       | `string`<br />The background color of the pin shape. Supports any CSS value. |
| `borderColor` | -       | `string`<br />The border color of the pin shape. Supports any CSS value.     |
| `glyph`       | -       | `string \| JSX.Element`<br />The glyph or DOM element to display in the pin. |
| `glyphColor`  | -       | `string`<br />The color of the glyph. Supports any CSS value.                |
| `scale`       | -       | `number`<br />The scale of the pin.                                          |
